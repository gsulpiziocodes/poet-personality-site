## 3) 14-Day Rollout Checklist

## Goals for 14 days
- Ship clear messaging
- Drive first qualified traffic
- Capture emails
- Get first paid signal (or waitlist intent)
- Identify best-performing channel

## Tracking setup (do Day 1)
Track these events:
- homepage_view
- analyze_started
- analyze_completed
- lead_captured
- account_created
- return_visit
- checkout_started (if monetization enabled)
- purchase_completed (if monetization enabled)

Primary KPIs:
- Visitor -> Analyze Start %
- Analyze Start -> Complete %
- Complete -> Email Capture %
- Email -> Return Visit %

---

### Day 1 — Messaging lock
- Finalize homepage copy blocks
- Update hero + trust framing
- Confirm CTA language consistency site-wide

### Day 2 — Funnel polish
- Remove friction from analyze flow
- Add clear “what happens next” after result
- Ensure capture form works reliably

### Day 3 — Analytics sanity
- Fire test events end-to-end
- Validate dashboard/event logging
- Snapshot baseline metrics

### Day 4 — Launch content prep
- Prepare 10 launch posts
- Create 3 visuals (hero, type card, result sample)
- Draft one short founder story post

### Day 5 — Soft launch (warm network)
- Share with friends/creators/writers
- Ask for 3 things: clarity, accuracy, confusion points
- Collect direct quotes as social proof

### Day 6 — Product iteration day
- Fix top 3 friction issues from soft launch
- Tighten unclear copy
- Improve result readability

### Day 7 — Public launch wave 1
- Post to X/IG/TikTok/LinkedIn (where relevant)
- Share one core thread/post + one short clip
- Respond to every comment

### Day 8 — Community distribution
- Share in 2–3 writing communities (no spam)
- Lead with value and context, not “check my app”
- Offer to analyze examples / explain framework

### Day 9 — SEO groundwork
- Publish /types and /categories intros with keyword-aware copy
- Ensure metadata + OG tags are set
- Index key pages

### Day 10 — Conversion tuning
- A/B test hero CTA text (Analyze Your Poems vs Find My Type)
- A/B test capture headline
- Keep winner after 24–48h

### Day 11 — Social proof pass
- Add 2–3 user quotes/screenshots
- Add “how many profiles analyzed” counter if credible
- Add trust disclaimer prominently

### Day 12 — Monetization test (light)
- Add waitlist or low-friction paid test (e.g., Deep Dive Report)
- Track checkout start intent even if payment not fully live

### Day 13 — Retention loop
- Send first nurture email:
  - your type result reminder
  - one tailored writing prompt
  - return CTA

### Day 14 — Review + next sprint plan
- Review KPI funnel
- Pick top channel
- Decide next 14-day focus:
  - acquisition
  - conversion
  - monetization
  - retention

## Quick “Done Means Done” Criteria
- Homepage messaging is clear in <10 seconds
- Users can submit -> result -> save without confusion
- You can explain Versiq in one sentence on camera
- You have one repeatable traffic channel
- You have at least one validated intent signal (email capture and/or payment intent)
